# RPSLS — Rock Paper Scissors Lizard Spock (Phaser 3)

**Short description**  
A single-page, responsive Phaser 3 web game implementing Rock, Paper, Scissors, Lizard, Spock with animated avatars, score-tracking, difficulty levels, simple AI with pattern recognition, and accessibility considerations.

## Files
- `index.html` — main page
- `styles.css` — styling and layout
- `game.js` — Phaser 3 game logic (ES module)
- `README.md` — this file

## Requirements
- A static web server. For local testing, use `live-server`, `http-server`, or VS Code Live Server.
- Modern browser with ES Modules support (Chrome, Firefox, Edge, Safari recent versions).

## How to run locally
1. Clone or download the project folder.
2. Start a static server in the project folder:
   ```bash
   # using node http-server
   npm install -g http-server
   http-server -c-1
   # or using live-server
   npx live-server
