# Rock, Paper, Scissors, Lizard, Spock Game

## Description
A one-page interactive game where users play against the computer. Features include animations, score tracking, and responsive design for mobile, tablet, and desktop.

## Features
- Pattern matching functionality
- Limited tries (optional)
- Score-tracking system
- Play against the computer
- Responsive design
- Engaging animations

## Technologies
- HTML5
- CSS3
- JavaScript (ES6)
- Responsive Design

## Deployment
- Can be hosted on GitHub Pages, Netlify, or other static hosting services.

## How to Play
1. Click on Rock, Paper, Scissors, Lizard, or Spock.
2. The computer will make its choice.
3. The winner is displayed and the score is updated.
